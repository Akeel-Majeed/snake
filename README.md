# Snake

A retro snake game built for the web.

## Quick Start

```bash
git clone https://github.com/Akeel-Majeed/snake.git
cd snake
npm install
npm run dev      # opens at http://localhost:5173
```

## Controls

| Action | Keys |
|--------|------|
| Move | Arrow keys or WASD |
| Pause / Resume | P or Esc |
| Mute | M |
| Confirm / Restart | Enter |
| Touch | Swipe in any direction |

## Commands

| Command | What it does |
|---------|--------------|
| `npm run build` | Production build → `dist/` |
| `npm run preview` | Preview the production build |
| `npm test` | Run all tests |
| `npm run coverage` | Tests + coverage report |

## Tech Stack

Vite · Vanilla JavaScript · HTML5 Canvas · Vitest
